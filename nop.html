NopFa

An online Shopping App Base on NopCommerce

nopCommerce is an open-source E-commerce solution based on Microsoft's ASP.NET Core framework and MS SQL Server 2012 (or higher) backend Database. It provides a catalog frontend and an administration tool backend, allowing shopping cart creation. It is available under the nopCommerce Public License V3 and officially launched in October 2008 for small to medium-sized businesses.

nopCommerce can be downloaded, installed and used free of charge. The community forum provides free support. There is an optional fee for white-labeling, premium support services, and partnership program. Until 2014, the documentation was downloaded on a paid basis and now is available free of charge.
